以 n-back task 探討視角改變對空間記憶能力之可能影響

Exploring The Possible Impact of Changes in Viewpoint on Visuo-spatial Working Memory with N-back Task

  • 研究目的
  • 實驗流程
  • 實驗說明

研究目的

本測驗使用 n-back task，依序呈現相同視角或不同視角的方格堆疊圖片刺激，欲了解受試者在視角改變下，對於空間記憶能力的可能影響。

  • 知情同意書

實驗流程

  • 實驗前準備 受試者繳交知情同意書，
    並以email寄送一式兩份之副本以供留存；
    研究人員講解實驗內容
  • 練習確認受試者對於作業內容是否有任何疑問
  • 正式實驗 共有四個階段，
    其間提供休息時段，
    受試者可自行決定時長
  • 實驗後回饋 填寫表單，
    留下心得與建議

實驗說明

1-back task

請將您所看見的刺激與 前一個 刺激相比，
若兩者相同，請按下鍵盤上的 f 鍵 ；
若不同，請按下 j 鍵 。

以圖片為例：當看見刺激"t"時，應與前一個刺激"T"相比。

2-back task

請將您所看見的刺激與 前兩個 刺激相比，
若兩者相同，請按下鍵盤上的 f 鍵 ；
若不同，請按下 j 鍵 。

以圖片為例：當看見刺激"B"時，應與前面第二個刺激"b"相比。

損害補償或保險

本研究依計畫執行，實驗過程不具有任何的危險性，
因此本研究不提供任何形式之補償。
